## 🚀 完成后的部署步骤

### 方法1：使用一键脚本（最简单）

```bash
cd ~/Desktop/vip-diamond-website

# 首次需要关联 GitHub 仓库
# 1. 在 GitHub 创建新仓库: https://github.com/new
# 2. 关联仓库
git remote add origin https://github.com/你的用户名/你的仓库名.git

# 3. 运行部署脚本
bash deploy.sh
```

### 方法2：分步操作

```bash
cd ~/Desktop/vip-diamond-website

# Step 1: 初始化 Git
git init
git add .
git commit -m "Initial commit"

# Step 2: 推送到 GitHub
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main

# Step 3: 部署到 Vercel
vercel
```

### 方法3：纯网页操作

1. **上传到 GitHub**
   - 访问 https://github.com/new
   - 创建新仓库
   - 上传 vip-diamond-website 文件夹中的所有文件

2. **部署到 Vercel**
   - 访问 https://vercel.com
   - 用 GitHub 登录
   - 导入刚才的仓库
   - 点击 Deploy
